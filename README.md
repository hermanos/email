email
=====

Description:
============
	 Aplicatia are rolul de a facilita scrierea si citirea emailului pentru persoanele cu deficiente de vedere.
Features:
=========
	- localizare
	- calendar
	- informatii
	- citire ceas
	- contact de urgenta
	- descriere imagine 

Requirments
===========
	Users pot trimite instant messages
	Users pot trimite email
	Users pot primi email
